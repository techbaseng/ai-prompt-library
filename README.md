# ✦ AI Prompt Library

> A curated, searchable library of **217 AI prompts** across **11 categories** — fully editable, one-click copy, works offline.

🔗 **Live site:** [babatundeawo.github.io/ai-prompt-library](https://babatundeawo.github.io/ai-prompt-library/)

---

## ✨ Features

| Feature | Description |
|---|---|
| 🔍 **Instant search** | Filter across all 217 prompts by keyword, category, or source |
| 📂 **11 categories** | Sidebar navigation + bottom tab bar on mobile |
| ✏️ **Editable prompts** | Click any textarea and fill in your `[PLACEHOLDERS]` directly |
| ↩️ **Reset button** | Restore any prompt to its original text in one click |
| 📋 **One-click copy** | Copy the full edited prompt to your clipboard instantly |
| ⭐ **Favourites** | Star prompts to save them; toggle "Favourites only" view |
| 🔢 **Field counter** | Each card shows how many `[PLACEHOLDERS]` need filling |
| ↕️ **Sort options** | Default / A→Z / Most fields / Fewest fields |
| 📱 **Mobile-first** | Bottom tab navigation on mobile, sidebar on desktop |
| ⌨️ **Keyboard shortcuts** | Press `/` to focus search, `Escape` to close |
| 🎬 **Stagger animations** | Cards fade in with smooth staggered entrance |
| 🌐 **Offline ready** | Single HTML file — no server, no framework, no dependencies |

## 🚀 How to Use

1. **Browse or search** for the prompt you want
2. **Click the textarea** to start editing
3. **Replace** all `[PLACEHOLDER]` fields with your specific content
4. **Click Copy** — full prompt copied to clipboard instantly
5. **Paste** into ChatGPT, Claude, Midjourney, Grok, or any AI tool
6. Click **↺** to reset the prompt back to the original at any time

## 🗂️ Repository Structure

```
ai-prompt-library/
├── index.html        # The complete app — all 217 prompts, styles, and logic
├── README.md         # This file
└── .nojekyll         # Tells GitHub Pages to serve the file as-is
```

The entire library is a **single self-contained HTML file** with no external dependencies except Google Fonts.

---

## 🚀 Deployment (GitHub Pages)

1. Go to [github.com/new](https://github.com/new) → create a **public** repo named `ai-prompt-library`
2. Upload `index.html`, `README.md`, and `.nojekyll`
3. Go to **Settings → Pages → Branch: main / (root) → Save**
4. Wait 1–3 minutes → live at **https://babatundeawo.github.io/ai-prompt-library/**
