# ✦ AI Prompt Library

> A curated, searchable library of **166 AI prompts** across **10 categories** — fully editable, one-click copy, works offline.

🔗 **Live site:** [babatundeawo.github.io/ai-prompt-library](https://babatundeawo.github.io/ai-prompt-library/)

---

## ✨ Features

| Feature | Description |
|---|---|
| 🔍 **Instant search** | Filter across all 166 prompts by keyword, category, or source |
| 📂 **10 categories** | Sidebar navigation with colour-coded categories |
| ✏️ **Editable prompts** | Click any textarea and fill in your `[PLACEHOLDERS]` directly |
| ↩️ **Reset button** | Restore any prompt to its original text in one click |
| 📋 **One-click copy** | Copy the full edited prompt to your clipboard instantly |
| ⭐ **Favourites** | Star prompts to save them; filter to see only favourites |
| 🔢 **Field counter** | Each card shows how many placeholders need filling |
| ↕️ **Sort options** | Default / A→Z / Most fields / Fewest fields |
| 📱 **Fully responsive** | Sidebar on desktop, drawer on mobile — works on all screens |
| ⌨️ **Keyboard shortcuts** | Press `/` to focus search, `Escape` to close |
| 🌐 **Offline ready** | Single HTML file — no server, no framework, no dependencies |

## 🚀 Usage

### Use the live site
Visit **[babatundeawo.github.io/ai-prompt-library](https://babatundeawo.github.io/ai-prompt-library/)** — no install needed.

### Run locally
1. Clone or download this repo
2. Open `index.html` in any browser
3. That's it — no build step, no server required

```bash
git clone https://github.com/babatundeawo/ai-prompt-library.git
cd ai-prompt-library
open index.html
```

### How to use a prompt
1. **Browse or search** for the prompt you want
2. **Click the textarea** to start editing
3. **Replace** all `[PLACEHOLDER]` fields with your specific content
4. **Click Copy** to copy the full prompt to your clipboard
5. **Paste** into ChatGPT, Claude, Midjourney, or any AI tool
6. Click **↺** to reset the prompt back to the original at any time
